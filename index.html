HTML Table Reference

Table Tags
Tag Name Description
<table> Table Defines the wrapper element for all HTML tables.
<caption> Caption Defines the caption of the table.
<thead> Table Head Defines the set of rows making up the column headers.
<th> Table Header Defines a cell as the header.
<tbody> Table Body Defines the set of rows making up the body of a table.
<tr> Table Row Defines a row of cells in a table.
<td> Table Data Defines a cell of table that contains data.
<foot> Table Foot The set of rows defining the footer in a table.
Table Attributes
Attribute Name Description
colspan Column span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.
